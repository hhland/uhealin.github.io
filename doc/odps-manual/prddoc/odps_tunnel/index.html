ODPS TUNNEL¶

ODPS TUNNEL是一种支持多种数据交换的通用服务。您可以使用ODPS TUNNEL提供的接口(Java API)编写程序读写ODPS的表(Table)。当前写表只支持追加(Insert Into)模式。本文档包括以下内容：

  • Tunnel SDK简介
    • UploadSession
    • DownloadSession
  • Tunnel SDK 示例
    • 简单上传示例
    • 简单下载示例
    • 多线程上传示例
    • 多线程下载示例

Note

本章节只对TUNNEL SDK的使用方法作简单的介绍，关于TUNNEL SDK的详细说明请参考SDK Java Doc。
